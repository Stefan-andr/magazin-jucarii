# Magazin de Jucarii

## Descriere

Aplicatia reprezinta un magazin de jucarii in care utilizatorul poate gestiona produsele din magazin.

## Operatii

- adaugarea unei jucarii
- afisarea listei de jucarii
- cautarea unei jucarii
- modificarea stocului
- stergerea unei jucarii

## Scop

Scopul aplicatiei este simularea functionarii unui magazin si exersarea lucrului cu interfete utilizator.
